# Manuel d'Utilisateur - PixelVerse

Ce guide vous explique comment utiliser les fonctionnalités clés de l'application PixelVerse Studios.

## 1. Accès au site
Le site est accessible à l'adresse suivante : `https://pixelverse-website.fly.dev`

### Comptes de test (Simulation)
- **Joueur** : `joueur@pixelverse.com` / `Password123!`
- **Employé** : `employe@pixelverse.com` / `Password123!`
- **Administrateur** : `admin@pixelverse.com` / `AdminPass123!`

## 2. Fonctionnalités Joueur
- **Inscription/Connexion** : Créez votre compte en acceptant les conditions RGPD.
- **Création de Personnage** : Utilisez le générateur aléatoire ou choisissez manuellement les traits.
- **Équipement** : Une fois votre personnage validé, équipez-le d'accessoires (épées, boucliers, etc.).
- **Galerie** : Consultez les créations de la communauté et filtrez par genre ou pseudo.
- **Avis** : Laissez un commentaire sur votre expérience (sujet à modération).

## 3. Fonctionnalités Employé
- **Tableau de Bord** : Visualisez les personnages en attente de validation.
- **Modération** : Approuvez ou rejetez les personnages et les avis clients.
- **Contact** : Répondez aux demandes des clients via l'interface dédiée.

## 4. Fonctionnalités Administrateur
- **Gestion des Employés** : Créez de nouveaux comptes employés ou suspendez des comptes utilisateurs.
- **Gestion des Accessoires** : Ajoutez, modifiez ou supprimez des objets de la base de données.
- **Consultation des Logs** : Visualisez l'historique complet des actions administratives (stocké via MongoDB Atlas).
